# Changelog

## 0.1.0 — 2026-05-25

Initial release.

- Build, preview, and export `.asy` files
- Auto-build on save with configurable format
- Interactive preview panel (pan, zoom, rotate, fit)
- Animation support with automatic frame detection
- Build recipes for SVG, PDF, PNG, and 3D render presets
- Problems panel integration with clickable error links
- Document outline (symbols) provider
- Hover documentation for functions and colors
- Autocomplete for functions, modules, and colors
- Lint on save (opt-in)
- 12 Asymptote code snippets
- Status bar with recipe selector and build status
- Export to PDF, SVG, PNG, and EPS
- Kill running build command
